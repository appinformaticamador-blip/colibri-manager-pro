# Fichajes 8.1 · Cuadrante certero

- El historial carga el cuadrante compartido de Supabase de la semana exacta de cada fichaje.
- Cruce por fecha, día, empleado e identificador normalizado.
- Muestra la franja prevista y calcula puntualidad contra la hora inicial real.
- Rediseño completo de fichajes abiertos e historial para móvil.
- No requiere SQL nuevo.
