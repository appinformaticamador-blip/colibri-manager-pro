# Rentabilidad real

1. Ejecuta `sql/colibri_erp_rentabilidad_real.sql` en Supabase SQL Editor.
2. Despliega el proyecto web.
3. En Rentabilidad abre **Rentabilidad real**.
4. Añade gastos fijos con fecha de inicio y fin opcional, gastos variables cuando ocurran y el coste/hora real de cada empleado.
5. Registra cada pago de personal indicando periodo, fecha y forma de pago.

La cuenta de resultados descuenta el coste devengado por horas fichadas. Los pagos se muestran como control de tesorería y saldo, pero no se descuentan por segunda vez.
