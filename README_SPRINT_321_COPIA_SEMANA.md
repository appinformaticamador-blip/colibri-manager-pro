# Sprint 3.2.1 - Cuadrantes: reutilizar semana anterior

## Objetivo
Permitir crear el cuadrante de una semana copiando el de la semana anterior, para modificar solo los cambios necesarios.

## Incluye
- Navegación semana anterior / semana siguiente.
- Botón Copiar semana anterior.
- Botón Duplicar a semana siguiente.
- Botón Vaciar semana.
- Mantiene máximo 3 empleados por franja.
- Mantiene exportación WhatsApp e imagen.
- Mejora responsive móvil para mantener tabla horizontal.
