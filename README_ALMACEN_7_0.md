# Colibrí Almacén 7.0

- Stock teórico por artículo maestro.
- Entradas automáticas desde facturas confirmadas.
- Consumo automático desde ventas NUMIER y escandallos/vínculos.
- Inventarios reales y diferencias.
- Registro de mermas.
- Stock mínimo, cobertura y pedido recomendado.
- Exportación CSV del pedido.

Los recuentos, mínimos, stock inicial y mermas se guardan en el navegador mediante localStorage. Los datos base se leen de Supabase.
