# COLIBRÍ ERP PRO · RC 3.8.0

## Incluye
- Vinculación de una compra con varios artículos NUMIER.
- Corrección de contraste en modales y tarjetas.
- Tarjetas de facturas adaptadas a móvil sin textos cortados.
- Catálogo maestro con unidad base, merma y coste manual opcional.
- Escandallos por gramos, kilogramos, mililitros, litros o unidades.
- Coste dinámico por receta y raciones obtenidas.
- Márgenes conjuntos de venta directa y platos elaborados.

## Paso obligatorio
Ejecutar en Supabase SQL Editor:

`sql/colibri_erp_rc380_escandallos_vinculacion_multiple.sql`

No requiere volver a desplegar las Edge Functions.
