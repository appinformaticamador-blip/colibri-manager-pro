# Sprint 3.2.4 - Cuadrantes: botones visibles encima del cuadrante

## Objetivo
Corregir la pantalla móvil para que las acciones de trabajo estén siempre visibles antes de la tabla del cuadrante.

## Incluye
- Semana anterior / actual / siguiente.
- Copiar semana anterior.
- Duplicar a semana siguiente.
- Copiar WhatsApp.
- Copiar/descargar imagen.
- Vaciar semana.
- Botonera forzada por CSS con `display:block/grid !important`.

## Archivos modificados
- src/App.jsx
- src/styles.css

## Prueba
1. Abrir Cuadrantes en móvil.
2. Verificar que aparece la botonera encima de la tabla.
3. Probar copiar semana anterior y duplicar semana.
4. Probar copiar WhatsApp.
