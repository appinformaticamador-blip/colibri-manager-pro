# RC 3.3.8 - Cuadrantes layout y empleados reales

Correcciones:
- El modal de selección queda por encima de la botonera de Control semanal en PC y móvil.
- La botonera superior deja de pisar la selección de empleados.
- Cuadrantes usa como fuente principal los empleados activos del módulo Empleados.
- Si no hay empleados en Supabase, usa plantilla base de seguridad.
- Mantiene Supabase, CERRADO, máximo 4 empleados y arrastrar/duplicar.

No requiere SQL nuevo si ya está aplicada RC 3.3.2.
