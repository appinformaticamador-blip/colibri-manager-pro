# Sprint 3.0.5 · IA Histórica

## Incluye
- Motor de histórico profundo desde 2023 usando `numier_daily_sales`.
- Objetivo IA con crecimiento +10% sobre histórico comparable.
- Comparativa con:
  - mismos días de la semana,
  - mismo periodo del año anterior,
  - tendencia de últimos 90 días.
- Predicción de cierre usando curva intradía de días comparables.
- Panel “IA histórica” con datos disponibles: rango, años, días y tickets históricos.
- Aviso si hay poco histórico.

## Instalación
1. Copiar `src/App.jsx` y `src/styles.css` sobre el proyecto en rama `develop`.
2. Commit: `Sprint 3.0.5 IA histórica`.
3. Push.
4. Probar en beta.

No necesita SQL nuevo ni recompilar Colibrí Engine.
