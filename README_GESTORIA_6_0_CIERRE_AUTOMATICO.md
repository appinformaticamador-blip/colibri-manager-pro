# Colibrí Gestoría 6.0 · Cierre automático

Incluye:
- cierre mensual y trimestral guiado;
- revisión de días sin cierre, descuadres y formas de pago no identificadas;
- separación de ventas, anulaciones y gastos Numier;
- desglose de efectivo, tarjeta, otros y pagos mixtos;
- Excel multipestaña;
- PDF resumen;
- paquete ZIP para gestoría;
- cierre/reapertura visual del periodo;
- diseño móvil.

## Instalación
Sustituir los archivos incluidos, ejecutar `npm install` y después `npm run build`.

No requiere SQL nuevo ni modificar ColibriEngine.exe.
