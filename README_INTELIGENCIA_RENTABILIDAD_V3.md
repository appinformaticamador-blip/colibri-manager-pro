# Inteligencia y rentabilidad v3

## Incluye
- Rangos: hoy, ayer, 3, 7 y 30 días, mes, año y fecha concreta.
- Ranking por producto con unidades, ventas, coste, beneficio y margen.
- Detalle de producto al pulsarlo.
- Rentabilidad por desayuno, almuerzo, tarde y cena.
- Detalle de tickets, mesas y contenido al pulsar cada franja.
- Todos los artículos NUMIER aparecen en Márgenes editables.
- Coste manual para artículos sin factura ni escandallo.

## SQL obligatorio
Ejecutar en Supabase SQL Editor:
`sql/colibri_erp_inteligencia_costes_manuales.sql`
