# Colibrí 6.0 — Centro de Mando

Sustituye la entrada Servicio por un centro de mando en directo que integra ventas, cuentas abiertas, rentabilidad, alertas, calidad de datos y estado del sincronizador.

## Instalación
Reemplaza `src/App.jsx` y `src/styles.css`, o sube el proyecto completo. No requiere SQL nuevo ni cambios en ColibriEngine.exe.
