# RC 4.0.2 — Portal completo del empleado

`fichar.braseria-elcolibri.es` deja de ser una pantalla aislada y pasa a ser el espacio personal del trabajador.

Incluye acceso nombre + PIN, fichaje GPS/QR, estado de jornada, horas semanales, comparación con cuadrante, estimación a 7 €/h cerrada, resumen WhatsApp, estadísticas, puntualidad, solicitud de salida olvidada y cambio de PIN.

## Instalación
1. Ejecutar `sql/colibri_erp_rc402_portal_empleado.sql` en Supabase.
2. Subir todo el proyecto a `develop`.
3. Vercel ejecutará `npm run build`.
