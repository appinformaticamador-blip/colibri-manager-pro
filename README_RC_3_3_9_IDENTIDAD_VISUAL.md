# COLIBRÍ ERP PRO — RC 3.3.9 Identidad Visual

Cambio exclusivamente visual aplicado sobre RC 3.3.8 estable.

## Incluye
- Nueva navegación lateral profesional en escritorio.
- Navegación horizontal compacta en móvil.
- Paleta pastel tecnológica: azul, verde, amarillo, violeta y coral.
- Logotipo Colibrí integrado en cabecera y acceso.
- Tarjetas, botones, formularios y módulos unificados visualmente.
- Cuadrantes adaptado a la nueva identidad sin modificar su lógica.
- Responsive para PC, tablet y móvil.

## No modifica
- Supabase.
- SQL.
- Persistencia de cuadrantes.
- Fichajes.
- Lógica funcional de los módulos.

## Prueba
`npm run build` completado correctamente.
