# Colibrí 5.0 · Dashboard de dirección

## Incluye
- Ventas y beneficio real estimado del día.
- Margen bruto, margen real y objetivo diario.
- Cuenta de resultados diaria: productos, personal por cuadrante, gastos fijos y variables.
- Alertas operativas y económicas accionables.
- Comparación con el mismo día anterior y media de cuatro semanas.
- Estado de servicio: mesas, barra, pendiente y cuenta más antigua.
- Resumen diario con tickets, ticket medio, producto líder y cierre de caja.
- Previsión mensual de beneficio real basada en el ritmo acumulado.
- Accesos directos a TPV, Servicio, Inteligencia, Rentabilidad y Cuadrantes.

No requiere SQL nuevo ni cambios en ColibriEngine.
