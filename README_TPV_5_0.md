# Colibrí TPV 5.0

Incluye:
- comparación con el mismo día de la semana anterior;
- tarjetas de ventas, tickets y ticket medio con referencias;
- nueva pestaña Control;
- detección de anulados, gastos, tickets altos, tickets muy bajos y tickets sin contenido;
- acceso directo al ticket real desde cada incidencia;
- estado visible del cierre de caja;
- diseño adaptado a móvil.

No requiere SQL nuevo ni cambios en ColibriEngine.
