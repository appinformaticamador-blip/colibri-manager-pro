# RC 3.5.1 — Dashboard Navegable

## Objetivo
Convertir el Dashboard Ejecutivo en centro de mando y navegación contextual.

## Incluye
- Tarjetas KPI enlazadas a TPV y Servicio.
- Índice Colibrí y recomendación enlazados a Inteligencia.
- Estado por zonas enlazado al plano de Servicio.
- Resumen del día con enlaces a TPV, Fichajes y Servicio.
- Actividad reciente enlazada a la cuenta abierta concreta o al TPV.
- Barra de accesos rápidos para Plano, Barra, TPV, Cuadrantes e IA.
- Navegación con historial del navegador y botón Atrás.
- Indicadores visuales de interacción y accesibilidad por teclado.

## SQL
No requiere SQL nuevo.
