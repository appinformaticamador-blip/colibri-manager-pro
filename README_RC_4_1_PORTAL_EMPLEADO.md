# RC 4.1 — Colibrí People

## Despliegue
1. Ejecutar en Supabase SQL Editor: `sql/colibri_erp_rc410_portal_empleado.sql`.
2. Subir el proyecto completo a la rama `develop`.
3. Vercel ejecutará `npm run build`.
4. Abrir `fichar.braseria-elcolibri.es` en una ventana privada o borrar la caché/PWA antigua.

## Portal del empleado
- Acceso personal con empleado + PIN.
- Inicio de jornada con estado en tiempo real.
- Entrada/salida GPS y respaldo QR.
- Horas semanales y mensuales.
- Comparación con el cuadrante publicado.
- Estimación a 7 €/h de jornadas cerradas.
- Resumen por WhatsApp.
- Evolución personal.
- Solicitud de salida olvidada con motivo obligatorio.
- Cambio de PIN personal.
- Historial de solicitudes.
- Activación de notificaciones del navegador.
