# Sprint 3.2.7 - Cuadrantes desde cero

Módulo de cuadrantes reconstruido desde cero sobre claves nuevas de localStorage para evitar datos corruptos anteriores.

Incluye botones visibles arriba, navegación semanal, copiar semana anterior, duplicar a siguiente, copiar día, exportar WhatsApp, exportar imagen, vaciar semana, empleados propios, máximo 3 empleados por franja, resumen de horas y drag & drop en escritorio.

Rollback: volver al zip 3.2.6 o restaurar src/App.jsx y src/styles.css anteriores.
