# RC 3.3.10 — Contraste profesional

Mejora exclusivamente visual sobre RC 3.3.9.

- Texto normal: negro/gris muy oscuro.
- Títulos y elementos estructurales: morado oscuro.
- Recomendaciones e información: azul.
- Estados positivos: verde oscuro.
- Estados críticos y errores: rojo.
- Advertencias: ámbar.
- Se eliminan textos blancos sobre fondos claros.
- No modifica lógica, Supabase, Cuadrantes ni datos.
