# Colibrí 5.0 · Inteligencia de decisiones

Añade al módulo Inteligencia:
- Pulso del negocio 0-100.
- Comparación automática con el periodo anterior equivalente.
- Mapa de productos: estrellas de beneficio, margen bajo y costes pendientes.
- Ritmo y media diaria del periodo.
- Mantiene rentabilidad por franjas, ranking completo, configuración directa de costes e informe ejecutivo.

No requiere SQL nuevo ni cambios en ColibriEngine.
