# RC 3.4.1 — Servicio LIVE: contraste semántico

Corrección visual exclusiva del módulo Servicio LIVE PRO.

- Texto normal: negro/gris oscuro.
- Títulos: morado oscuro.
- Recomendaciones e información: azul.
- Estados positivos e importes: verde oscuro.
- Alertas críticas: rojo.
- Advertencias: ámbar.
- Se elimina texto blanco heredado sobre tarjetas claras.
- Se corrige especialmente Actividad actual, Lectura IA, paneles laterales y modal de ticket.

No modifica consultas, Supabase, NUMIER ni lógica del módulo.
No requiere SQL nuevo.
